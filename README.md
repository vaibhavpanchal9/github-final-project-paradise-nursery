# Paradise Nursery 🌿

Paradise Nursery is a dynamic e-commerce web application that allows users to browse and purchase a variety of houseplants.

## 🚀 Features
- Browse plants by category
- View plant images, names, descriptions, and prices
- Add plants to shopping cart
- Dynamic cart updates (increase, decrease, remove)
- Real-time total cost calculation
- Navigation bar for easy page access

## 🛠️ Technologies Used
- React.js
- Redux Toolkit
- CSS

## 🎯 Project Goal
To build a functional and user-friendly shopping cart system by applying front-end development concepts and state management.
